2. Visualización por materia#

Generar histogramas y box plots de la columna ‘Materia Modalidade’ para cada materia de modalidad para visualizar sus distribuciones.

import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np
import math

# Get unique modalities, sort alphabetically, and create a color mapping using 'Set3'
modalities = sorted(modalidad_df['Nom. Materia modalidade'].unique())
colors = sns.color_palette('Set3', len(modalities))
color_map = dict(zip(modalities, colors))

# Group the DataFrame by 'Nom. Materia modalidade'
grouped_modalidad_df = modalidad_df.groupby('Nom. Materia modalidade')['Materia Modalidade']

# Histograms
num_modalidades = len(modalities)
cols = 3
rows = math.ceil(num_modalidades / cols)

plt.figure(figsize=(cols * 5, rows * 4))

for i, modality in enumerate(modalities):
    plt.subplot(rows, cols, i + 1)
    group = grouped_modalidad_df.get_group(modality)
    # Create bins from 0 to 10 with a step of 1
    bins = np.arange(0, 11, 1)  # Bins from 0 to 10
    plt.hist(group, bins=bins, color=color_map[modality], edgecolor='black')  # Use color map
    plt.title(f'Distribución de Cualificacións para {modality}')
    plt.xlabel('Cualificación')
    plt.ylabel('Frecuencia')
    plt.xlim(0, 10)
    plt.ylim(0, 4)  # Set fixed y-axis limit
    plt.yticks(np.arange(0, 5, 1))  # Set fixed y-ticks

    plt.grid(axis='y', linestyle='--')

plt.tight_layout()
plt.show()

# Box plot with individual points and means
plt.figure(figsize=(10, 6))
sns.boxplot(data=modalidad_df, y='Nom. Materia modalidade', x='Materia Modalidade', hue='Nom. Materia modalidade', palette=color_map, order=modalities, legend=False)  # Use color map and specify order
sns.swarmplot(data=modalidad_df, y='Nom. Materia modalidade', x='Materia Modalidade', color='black', size=5, ax=plt.gca(), order=modalities)  # Add individual points and specify order

# Add mean points for each modality
modalities_mean = grouped_modalidad_df.mean().reindex(modalities)  # Reindex means to match sorted order
plt.scatter(y=range(len(modalities)), x=modalities_mean.values, color='white', edgecolors='black', s=100, zorder=3)  # Add white points with black edge

plt.title('Box Plot de Cualificacións por Materia de Modalidade con Puntos Individuais e Medias')
plt.ylabel('Materia de Modalidade')
plt.xlabel('Cualificación')
plt.xlim(0, 10)
plt.xticks(np.arange(0, 11, 1))  # Set x-ticks from 0 to 10 with step 1
plt.yticks(rotation=45, ha='right')

# Add horizontal grid lines
plt.grid(axis='x', linestyle='--')

plt.tight_layout()
plt.show()

3. Conclusiones#

Analizamos los resultados estadísticos y las visualizaciones para comparar el rendimiento en las diferentes materias de modalidad e identificar posibles diferencias.

3.1. Tendencias generales del rendimiento:#

  • La media global de las materias de modalidad es 6,84, y la mediana global es 6,75. → Esto indica un rendimiento medio-alto en conjunto, con ligeros sesgos positivos (la media está ligeramente por encima de la mediana).

  • La mayoría de las asignaturas parecen tener pocos alumnos, lo cual es habitual en materias de 2º de Bachillerato. Las calificaciones tienden a concentrarse entre el 5 y el 8, reflejando en general un nivel medio de rendimiento. Las materias con más alumnos permiten una mejor observación de la distribución y muestran una mayor diversidad de resultados.

3.2. Análisis por materia de modalidad#

1. Análisis Musical II#

  • Datos estadísticos:

    • Solo hay un dato (count = 1.0) con una calificación de 6.5.

    • No hay desviación estándar (std = NaN) porque solo hay un valor.

  • Histograma:

    • No se puede analizar por falta de datos suficientes.

  • Box Plot:

    • Se representa como una línea única en 6.5 (sin rango intercuartílico ni bigotes).

  • Conclusión:

    • Muestra insuficiente para análisis significativo. Podría tratarse de un error o una materia con muy pocos estudiantes.

2. Artes Escénicas II#

  • Datos estadísticos:

    • 4 estudiantes, media = 4.875, desviación estándar baja (0.250).

    • Rango estrecho: 4.5 a 5.0 (sin outliers).

  • Histograma:

    • Concentrado en 5.0, con una pequeña frecuencia en 4.5.

  • Box Plot:

    • La caja es estrecha (Q1 = 4.875, Q3 = 5.0), confirmando poca variabilidad.

    • No hay bigotes significativos, ya que todos los datos están muy agrupados.

  • Conclusión:

    • Rendimiento homogéneo y alto. Posible evaluación poco discriminante o grupo muy uniforme.

3. Debuxo Artístico II#

  • Datos estadísticos:

    • 9 estudiantes, media = 6.944, desviación estándar moderada (1.483).

    • Rango amplio: 5.0 a 9.25.

  • Histograma:

    • Distribución ligeramente sesgada a la izquierda, con mayoría en 6.0–8.0 y un caso bajo (5.0).

  • Box Plot:

    • Caja muestra Q1 = 6.0, Q3 = 8.0 (rango intercuartílico amplio).

    • Bigotes llegan a 5.0 (mínimo) y 9.25 (máximo).

    • Posible outlier en 5.0 (punto alejado de la caja).

  • Conclusión:

    • Mayoría con buen rendimiento, pero un estudiante rezagado. Podría requerir apoyo individualizado.

4. Matemáticas Apl. ás CC.SS.#

  • Datos estadísticos:

    • 10 estudiantes, media = 6.95, alta desviación estándar (1.825).

    • Rango muy amplio: 3.5 a 9.5.

  • Histograma:

    • Distribución bimodal o con outliers (picos en extremos bajos y altos).

  • Box Plot:

    • Caja centrada en 6.06–8.06 (Q1–Q3), pero con bigotes hasta 3.5 y 9.5.

    • Outlier claro en 3.5 (punto fuera del bigote inferior).

  • Conclusión:

    • Gran variabilidad. Algunos estudiantes sobresalen (9.5), pero otros fracasan (3.5). Sugiere diferencias en comprensión o metodología de enseñanza.

5. Matemáticas II#

  • Datos estadísticos:

    • 6 estudiantes, media = 8.083, desviación estándar moderada (1.281).

    • Rango: 5.75 a 9.5.

  • Histograma:

    • Sesgo a la derecha, con mayoría en 7.5–9.5 y un caso bajo (5.75).

  • Box Plot:

    • Caja estrecha (Q1 = 7.875, Q3 = 8.687), pero con bigote largo hacia abajo (5.75).

    • Outlier en 5.75 (alejado del grupo principal).

  • Conclusión:

    • Excelente rendimiento general, pero un estudiante con dificultades. Podría ser un caso aislado que necesita refuerzo.

3.3. Conclusión General#

  1. Materias artísticas (Artes Escénicas II, Debuxo Artístico II):

    • Resultados más homogéneos, especialmente en Artes Escénicas II.

    • En Debuxo, hay un posible outlier que sugiere atención individual.

  2. Matemáticas (Matemáticas Apl. ás CC.SS. y Matemáticas II):

    • Matemáticas Apl. ás CC.SS. muestra gran dispersión, con estudiantes en extremos opuestos (éxito vs. fracaso).

    • Matemáticas II tiene un rendimiento alto en general, pero con un outlier bajo.

  3. Análisis Musical II:

    • Datos insuficientes para análisis.

3.4. Recomendaciones#

  • Matemáticas Apl. ás CC.SS.: Investigar causas de la dispersión (¿dificultad de la materia? ¿métodos de evaluación?).

  • Matemáticas II y Debuxo Artístico II: Revisar el caso del estudiante con calificación baja para ofrecer apoyo.

  • Artes Escénicas II: Evaluar si la falta de variabilidad se debe a una evaluación poco exigente o a un grupo homogéneo.

3.5. Patrón general#

  • Las materias técnicas (matemáticas) presentan mayor desigualdad en resultados, mientras que las artísticas son más consistentes. Esto podría reflejar diferencias en la forma de evaluación o en la preparación previa de los estudiantes.
