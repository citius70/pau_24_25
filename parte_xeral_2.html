Análisis 2: Análisis de resultados diferenciando materias de modalidad#

Es oportuno hacer un análisis de los datos teniendo en cuenta las distintas materias de modalidad.

Objetivo:

  • Identificar las materias de modalidad: Filtrar el DataFrame para incluir solo las columnas relevantes para las materias de modalidad.

  • Agrupar por materia de modalidad: Agrupar los datos por la columna ‘Materia modalidade’.

  • Análisis estadístico por materia: Calcular los estadísticos descriptivos (media, mediana, desviación estándar, etc.) para la columna ‘Cualif. Materia Modalidade’ dentro de cada grupo (cada materia de modalidad).

  • Visualización por materia: Generar histogramas y/o box plots de la columna ‘Cualif. Materia Modalidade’ para cada materia de modalidad para visualizar sus distribuciones.

  • Comparar y concluir: Analizar los resultados estadísticos y las visualizaciones para comparar el rendimiento en las diferentes materias de modalidad e identificar posibles diferencias.

2. Visualización por materia#

Generar histogramas y box plots de la columna ‘Materia Modalidade’ para cada materia de modalidad para visualizar sus distribuciones.

import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np
import math

# Get unique modalities and create a color mapping using 'Set3'
modalities = modalidad_df['Nom. Materia modalidade'].unique()
colors = sns.color_palette('Set3', len(modalities))
color_map = dict(zip(modalities, colors))

# Histograms
num_modalidades = len(modalities)
cols = 3
rows = math.ceil(num_modalidades / cols)

plt.figure(figsize=(cols * 5, rows * 4))

for i, modality in enumerate(modalities):
    plt.subplot(rows, cols, i + 1)
    group = grouped_modalidad_df.get_group(modality)
    plt.hist(group, bins=10, color=color_map[modality]) # Use color map
    plt.title(f'Distribución de Cualificacións para {modality}')
    plt.xlabel('Cualificación')
    plt.ylabel('Frecuencia')
    plt.xlim(0, 10)
    plt.ylim(0, 4) # Set fixed y-axis limit
    plt.yticks(np.arange(0, 5, 1)) # Set fixed y-ticks

    plt.grid(axis='y', linestyle='--')

plt.tight_layout()
plt.show()

# Box plot with individual points and means
plt.figure(figsize=(10, 6))
sns.boxplot(data=modalidad_df, x='Nom. Materia modalidade', y='Materia Modalidade', hue='Nom. Materia modalidade', palette=color_map, legend=False) # Use color map
sns.swarmplot(data=modalidad_df, x='Nom. Materia modalidade', y='Materia Modalidade', color='black', size=5, ax=plt.gca()) # Add individual points

# Add mean points for each modality
modalities_mean = grouped_modalidad_df.mean()
plt.scatter(x=range(len(modalities)), y=modalities_mean.values, color='white', edgecolors='black', s=100, zorder=3) # Add white points with black edge

plt.title('Box Plot de Cualificacións por Materia de Modalidade con Puntos Individuais e Medias')
plt.xlabel('Materia de Modalidade')
plt.ylabel('Cualificación')
plt.ylim(0, 10)
plt.xticks(rotation=45, ha='right')



plt.legend()
plt.tight_layout()
plt.show()

3. Conclusiones#

Analizamos los resultados estadísticos y las visualizaciones para comparar el rendimiento en las diferentes materias de modalidad e identificar posibles diferencias.

3.1. Análisis de los histogramas#

  1. Asignaturas con pocos alumnos:

    • Análisis Musical II, Latín II y Matemáticas II muestran distribuciones con solo 1 o 2 valores diferentes, lo que indica que muy pocos estudiantes están matriculados en esas materias (posiblemente optativas de baja demanda).

    • Esto limita la capacidad de análisis estadístico, pero sugiere cierto nivel de homogeneidad en el rendimiento.

  2. Asignaturas con mayor número de estudiantes:

    • Artes Escénicas II tiene tres calificaciones en torno al 5, lo que podría sugerir un rendimiento ajustado al aprobado, aunque con escasa variedad.

    • Debuxo Artístico II presenta una distribución más completa con varias notas entre 5 y 9. La mayoría de estudiantes se concentra alrededor del 6, lo que sugiere un rendimiento medio.

    • Matemáticas Aplicadas a las Ciencias Sociales muestra una distribución bastante amplia, desde el 4 hasta el 9, con una concentración en los valores de 7 y 8. Esto indica un grupo más numeroso y un rendimiento general bueno, aunque con cierta dispersión.

  3. Comparación de rendimiento:

    • Matemáticas II (modalidad científica) presenta solo tres calificaciones: 6, 8 y 9, todas aprobadas y en el rango medio-alto. Esto sugiere un grupo pequeño pero con buen rendimiento.

    • En cambio, en Matemáticas Aplicadas, aunque también hay buenos resultados, hay más dispersión y presencia de calificaciones más bajas, como el 4.

🧠 Reflexión final:#

La mayoría de las asignaturas parecen tener pocos alumnos, lo cual es habitual en materias optativas de 2º de Bachillerato. Las calificaciones tienden a concentrarse entre el 5 y el 8, reflejando en general un nivel medio de rendimiento. Las materias con más alumnos permiten una mejor observación de la distribución y muestran una mayor diversidad de resultados.

3.2. Conclusiones a partir del Boxplot y los estadísticos descriptivos:#

1. Tendencias generales del rendimiento:#

  • La media global de las materias de modalidad es 6,84, y la mediana global es 6,75. → Esto indica un rendimiento medio-alto en conjunto, con ligeros sesgos positivos (la media está ligeramente por encima de la mediana).

2. Materias con mejor rendimiento:#

  • Matemáticas II:

    • Media: 8,08; Mediana: 8,375

    • Tiene una dispersión moderada (std 1,28) y no presenta calificaciones por debajo del 5.

    • Es la materia con mejor media, y los resultados están por encima de la media general.

  • Matemáticas Aplicadas ás CC.SS.:

    • Media: 6,95; Mediana: 7,25

    • Tiene la mayor desviación estándar (1,82), lo que indica alta dispersión: hay estudiantes con calificaciones muy altas (hasta 9,5) y otros más bajos (mínimo de 3,5).

    • Aunque el rendimiento medio es bueno, es la materia con más variabilidad entre estudiantes.

3. Materias con rendimiento medio:#

  • Debuxo Artístico II:

    • Media: 6,94, muy cercana a la media global.

    • Amplio rango de notas (de 5 a 9,25) y dispersión moderada (std 1,48).

    • Su mediana está ligeramente por debajo de su media, lo que sugiere asimetría positiva (algunos estudiantes con notas altas elevan la media).

  • Latín II:

    • Media: 6,25, mediana igual.

    • Dispersión baja (std 1,06), con calificaciones entre 5,5 y 7.

    • Rendimiento estable, aunque ligeramente inferior a la media general.

4. Materias con rendimiento más bajo:#

  • Artes Escénicas II:

    • Media: 4,87, la más baja de todas las materias.

    • Mediana: 5, con una desviación mínima (std 0,25).

    • Muy poca variabilidad, pero todos los valores están en el umbral del aprobado o por debajo. → Puede indicar dificultades del alumnado o exigencias de evaluación elevadas.

5. Casos especiales:#

  • Análisis Musical II:

    • Solo tiene una calificación (6,5), por lo tanto no presenta dispersión.

    • No es posible hacer comparaciones estadísticas con otras materias.

    • Su valor se sitúa ligeramente por debajo de la media general.

📌 Reflexión global:#

  • La mayoría de las materias de modalidad presentan un rendimiento satisfactorio, por encima del aprobado.

  • Destaca el excelente rendimiento en Matemáticas II, y la buena progresión en Matemáticas Aplicadas, aunque esta última requiere seguimiento por su alta dispersión.

  • Artes Escénicas II merece atención especial, ya que presenta los peores resultados globales.

  • El número reducido de estudiantes en algunas materias limita la validez estadística, pero permite identificar tendencias.

import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np
import math

# Histograms
num_modalidades = len(grouped_modalidad_df)
cols = 3
rows = math.ceil(num_modalidades / cols)

plt.figure(figsize=(cols * 5, rows * 4))
for i, (modality, group) in enumerate(grouped_modalidad_df):
    plt.subplot(rows, cols, i + 1)
    counts, bins, patches = plt.hist(group, bins=10)
    plt.title(f'Distribución de Cualificacións para {modality}')
    plt.xlabel('Cualificación')
    plt.ylabel('Frecuencia')
    plt.xlim(0, 10)
    plt.yticks(np.arange(0, int(max(counts)) + 1, 1))
    plt.grid(axis='y', linestyle='--')

plt.tight_layout()
plt.show()

# Box plot
plt.figure(figsize=(10, 6))
sns.boxplot(data=modalidad_df, x='Nom. Materia modalidade', y='Materia Modalidade', hue='Nom. Materia modalidade', palette='viridis', legend=False)
plt.title('Box Plot de Cualificacións por Materia de Modalidade')
plt.xlabel('Materia de Modalidade')
plt.ylabel('Cualificación')
plt.ylim(0, 10) # Adjusted y-limit for better visualization
plt.xticks(rotation=45, ha='right')

# Add overall mean and median lines
overall_mean = modalidad_df['Materia Modalidade'].mean()
plt.axhline(overall_mean, color='red', linestyle='dashed', linewidth=1, label=f'Media Materias Modalidade ({overall_mean:.2f})')

overall_median = modalidad_df['Materia Modalidade'].median()
plt.axhline(overall_median, color='green', linestyle='dashed', linewidth=1, label=f'Mediana Materias Modalidade ({overall_median:.2f})')

plt.legend()
plt.tight_layout()
plt.show()
