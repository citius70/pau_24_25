Análisis descriptivo PAU 2024-25#

IES FREI MARTÍN SARMIENTO (PONTEVEDRA)#

Objetivo:

  • Realizar un estudio estadístico descriptivo sobre los datos del archivo.

  • Analizar las variables numéricas del conjunto de datos para comprender su distribución, tendencia central, dispersión y posibles valores atípicos.

Herramientas:

  • Lenguaje de programación: Python

  • Librerías de Python: Pandas, Numpy y Matplotlib

Pasos realizados:

  1. Estadísticos descriptivos básicos:

    • Se calcularon medidas como la media, mediana, desviación estándar, mínimo, máximo y cuartiles para cada columna numérica.

    • Esto permite evaluar la tendencia central y la variabilidad de los datos.

  2. Visualización de distribuciones:

    • Histogramas: Para observar la forma de la distribución (normal, sesgada, bimodal, etc.).

    • Diagramas de caja (box plots): Para identificar valores atípicos y comparar la dispersión entre variables.

  3. Análisis de valores faltantes:

    • Se verificó la presencia de datos faltantes o inconsistentes que puedan afectar el análisis.

Resultados esperados:

  • Identificar qué variables siguen una distribución normal y cuáles presentan sesgos.

  • Detectar posibles errores o anomalías en los datos (valores extremos o faltantes).

  • Determinar si algunas columnas son categóricas (como códigos) en lugar de numéricas.

Ejemplo de interpretación:
Si una columna como Cualificación parte xeral muestra una media y mediana similares con poca desviación estándar, su distribución sería cercana a la normal. En cambio, si Historia de España tiene una mediana muy distinta de la media con una cola larga en el histograma, indicaría un sesgo, posiblemente por valores faltantes o concentración en extremos.

Este estudio proporciona una base para futuros análisis, como comparaciones entre grupos o modelos predictivos.

3. Visualización de los datos#

Generamos visualizaciones para explorar la distribución de los datos. Los gráficos seleccionados son histogramas y diagramas de caja (box plot).

import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np

# Select numerical columns for visualization
numerical_cols = df.select_dtypes(include=['number'])
columns_to_exclude = ['Cod. Lingua Estranxeira', 'Cod. Materia Modalidade'] # Corrected column name
numerical_cols_viz = numerical_cols.drop(columns=columns_to_exclude)

# Calculate the mean excluding the last two columns
mean_excluding_last_two = numerical_cols_viz.iloc[:, :-2].mean().mean()

# Calculate the overall mean of all numerical columns being visualized
overall_mean_numerical_viz = numerical_cols_viz.mean().mean()


# Create histograms
numerical_cols_viz.hist(bins=10, figsize=(15, 10))
plt.suptitle('Histogramas de Columnas Numéricas', y=0.92, fontsize=16)
plt.tight_layout(rect=[0, 0.03, 1, 0.95])
plt.grid(axis='y', linestyle='--') # Add horizontal grid lines
plt.xlim(0, 10) # Set x-axis limits from 0 to 10
plt.show()

# Create box plots
plt.figure(figsize=(10, 8))
sns.boxplot(data=numerical_cols_viz)
plt.title('Box Plots de Columnas Numéricas')
plt.xticks(rotation=45, ha='right')
plt.yticks(np.arange(0, 11, 1)) # Set y-axis ticks from 0 to 10 with step 1
plt.ylim(0, 10) # Set y-axis limits from 0 to 10
plt.xlabel('MATERIAS') # Set x-axis label
plt.ylabel('PUNTUACIONES: Mínimo, 2º cuartil, media, 3º cuartil, máximo') # Set y-axis label

# Add horizontal line for the mean excluding the last two columns
plt.axhline(mean_excluding_last_two, color='red', linestyle='dashed', linewidth=1, label=f'Media (excluyendo últimas 2 columnas) ({mean_excluding_last_two:.2f})')

# Add horizontal line for the overall mean of all numerical columns being visualized
plt.axhline(overall_mean_numerical_viz, color='blue', linestyle='dotted', linewidth=1, label=f'Media General ({overall_mean_numerical_viz:.2f})')


plt.legend() # Show the legend for the horizontal lines
plt.tight_layout()
plt.show()

  • Conclusión: Predominan también las notas entre 5 y 7, aunque hay mayor variabilidad. Algunos estudiantes destacan con notas altas.

🔹 3. Historia de España#

  • Distribución: Más dispersa, con un número significativo de notas bajas (3-5) y altas (8-9).

🔹 4. Historia da Filosofía#

🔹 6. Cualif. Materia Modalidade#

  • Distribución: También bastante equilibrada, pero con mayor dispersión que en lengua extranjera.

🔹 7. Cualificación parte xeral#

import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np
import math # Import math for calculating grid size

# Create subplots for each modality subject's histogram
num_modalities = len(grouped_modalidad_df)
cols = 3  # Number of columns in the subplot grid
rows = math.ceil(num_modalities / cols) # Calculate the number of rows needed

plt.figure(figsize=(cols * 5, rows * 4)) # Adjust figure size based on grid size
for i, (modality, group) in enumerate(grouped_modalidad_df):
    plt.subplot(rows, cols, i + 1) # Select the current subplot
    counts, bins, patches = plt.hist(group['Cualif. Materia Modalidade'], bins=10) # Capture histogram data
    plt.title(f'Distribución de Cualificacións para {modality}')
