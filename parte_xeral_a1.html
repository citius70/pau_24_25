A1. Proba xeral: sin diferenciar materias de modalidad#

Objetivo:

  • Realizar un estudio estad√≠stico descriptivo sobre los datos del archivo proba_xeral

  • Analizar las variables num√©ricas del conjunto de datos para comprender su distribuci√≥n, tendencia central, dispersi√≥n y posibles valores at√≠picos.

Herramientas:

  • Lenguaje de programaci√≥n: Python

    • Librer√≠as de Python: Pandas, Numpy, Seaborn y Matplotlib

Pasos realizados:

  1. Estad√≠sticos descriptivos b√°sicos:

    • Se calcularon medidas como la media, mediana, desviaci√≥n est√°ndar, m√≠nimo, m√°ximo y cuartiles para cada columna num√©rica.

    • Esto permite evaluar la tendencia central y la variabilidad de los datos.

  2. Visualizaci√≥n de distribuciones:

    • Histogramas: Para observar la forma de la distribuci√≥n (normal, sesgada, bimodal, etc.).

    • Diagramas de caja (box plots): Para identificar valores at√≠picos y comparar la dispersi√≥n entre variables.

  3. An√°lisis de valores faltantes:

    • Se verific√≥ la presencia de datos faltantes o inconsistentes que puedan afectar el an√°lisis.

Resultados esperados:

  • Identificar qu√© variables siguen una distribuci√≥n normal y cu√°les presentan sesgos.

  • Detectar posibles errores o anomal√≠as en los datos (valores extremos o faltantes).

  • Determinar si algunas columnas son categ√≥ricas (como c√≥digos) en lugar de num√©ricas.

Ejemplo de interpretaci√≥n:

Si una columna muestra una media y mediana similares con poca desviaci√≥n est√°ndar, su distribuci√≥n ser√≠a cercana a la normal. En cambio, si tiene una mediana muy distinta de la media con una cola larga en el histograma, indicar√≠a un sesgo, posiblemente por valores faltantes o concentraci√≥n en extremos.

Este estudio proporciona una base para futuros an√°lisis, como comparaciones entre grupos o modelos predictivos.

3. Visualizaci√≥n de los datos#

Generamos visualizaciones para explorar la distribuci√≥n de los datos. Los gr√°ficos seleccionados son histogramas y diagramas de caja (box plot).

import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np
import math

# Seleccionar columnas num√©ricas
num_cols = df.select_dtypes(include=['number'])
cols_excluir = ['Cod. Lingua Estranxeira', 'Cod. Materia Modalidade']
num_cols_viz = num_cols.drop(columns=cols_excluir)

# Eliminar columnas con todos los valores NaN
num_cols_viz = num_cols_viz.dropna(axis=1, how='all')

# Calcular las medias
means = num_cols_viz.mean()

# Definir los bins para los histogramas
bins = np.arange(0, 11, 1)

# Crear una paleta de colores
palette = sns.color_palette("Set1", n_colors=num_cols_viz.shape[1])

# Crear histogramas
fig, axes = plt.subplots(nrows=math.ceil(num_cols_viz.shape[1]/3), ncols=3, figsize=(15, 10))
axes = axes.flatten()

for i, col in enumerate(num_cols_viz.columns):
    axes[i].hist(num_cols_viz[col].dropna(), bins=bins, color=palette[i], edgecolor='black')
    axes[i].set_title(col)
    axes[i].set_xlabel('Cualificaci√≥n')
    axes[i].set_ylabel('Frecuencia')
    axes[i].set_xlim(0, 10)
    axes[i].set_ylim(0, 14)
    axes[i].set_xticks(np.arange(0, 11, 1))
    axes[i].set_yticks(np.arange(0, 15, 1))
    axes[i].grid(axis='y', linestyle='--')

for j in range(i + 1, len(axes)):
    axes[j].set_visible(False)

plt.suptitle('Histogramas: Distribuci√≥n de las calificaciones', y=0.95, fontsize=16)
plt.tight_layout(rect=[0, 0.03, 1, 0.95])
plt.show()

# Crear boxplot horizontal
plt.figure(figsize=(15, 10))
sns.boxplot(data=num_cols_viz, palette=palette, orient='h')  # Cambiar a orientaci√≥n horizontal
sns.swarmplot(data=num_cols_viz, color=".25", orient='h')  # Asegurarse de que el swarmplot tambi√©n sea horizontal

# A√±adir las medias al boxplot
plt.scatter(x=means, y=range(num_cols_viz.shape[1]), color='white', s=50, zorder=3)

# L√≠nea vertical para separar las materias
plt.axhline(y=len(num_cols_viz.columns) - 2.5, color='black', linestyle='--', linewidth=1)

plt.title('Box Plots con calificaciones individuales')
plt.yticks(rotation=45, ha='right')
plt.xticks(np.arange(0, 11, 1))
plt.xlim(0, 10)
plt.xlabel('M√≠nimo, 2¬∫ cuartil, media, 3¬∫ cuartil, m√°ximo')
# plt.ylabel('MATERIAS')

plt.tight_layout()
plt.show()
